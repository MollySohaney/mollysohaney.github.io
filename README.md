# Molly Sohaney Portfolio

Personal portfolio site for Molly Sohaney, built for GitHub Pages.

## Site Structure

- Home
- About
- Projects
- Writeups
- Contact

## Content Highlights

- Cybersecurity-focused homepage with resume links and featured work
- About page with education, skills, certifications, experience, and clubs
- Projects page with public and academic project cards
- Writeups page for The Flag Vault and CTF-related content
